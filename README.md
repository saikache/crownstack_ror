# README

ruby 2.5.3
rails 5.2.4

User creation
  POST: /api/users

User login
  POST: /api/auth/login

To list all the products
  GET: /api/products

Add a product to logged in user's cart
  POST: /api/products/:id/add_to_cart

Display logged in user's cart
  GET: /api/users/cart
